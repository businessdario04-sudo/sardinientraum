Anfragen & E-Mail-Empfang

Wohin sollen Buchungsanfragen vom Formular gesendet werden?

So funktioniert es: Der kostenlose Dienst Formspree.io leitet Formular-Einsendungen direkt an deine E-Mail weiter — ohne eigenen Server.

1. Kostenlos registrieren auf formspree.io
2. Neues Formular anlegen → deine E-Mail-Adresse eingeben
3. Die Formular-ID (z.B. xpwzgkla) hier eintragen
4. „Auf Website anwenden" klicken — fertig!
Formspree-ID Aus deiner Formspree-URL: formspree.io/f/HIER
Empfänger-E-Mail Zur Kontrolle – muss mit Formspree übereinstimmen
Betreff-Vorlage Wie Anfragen in deinem Postfach erscheinen
Änderungen werden in der Website gespeichert.
